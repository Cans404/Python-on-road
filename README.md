2、ip_zip_unzip.py

IP变换算法，比如IP地址1.1.1.129具体规则如下：

1、转化为二进制：00000001000000010000000110000001\n 
2、替换（0用a表示，1用b表示）：aaaaaaabaaaaaaabaaaaaaabbaaaaaab\n 
3、将连续相同的字符合并：7a1b7a1b7a2b6a1b\n 
4、减少长度，将数字1去掉：7ab7ab7a2b6ab\n
